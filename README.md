# IndustrialTransformer

## Install

### Install with pip

#### CPU

```bash
uv pip install ".[cpu]"
```

#### GPU

```bash
uv pip install ".[cuda]"
```

### Install with `-e`

#### CPU

```bash
uv pip install -e ".[cpu,dev]"
```

#### GPU

```bash
uv pip install -e ".[cuda,dev]"
```

#### M Series

```bash
uv pip install -e ".[m,dev]"

```
