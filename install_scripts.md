# Install

## GPU

```bash
pip install --upgrade pip
pip install torchtext
pip install --upgrade "jax[cuda12_pip]" -f https://storage.googleapis.com/jax-releases/jax_cuda_releases.html
pip install flax
pip install polars
pip install spacy
```

## CPU

```bash
pip install --upgrade pip
pip install torchtext
pip install --upgrade "jax[cpu]"
pip install flax
pip install polars
pip install spacy
```
